# YOLO

Denne dokumentation er ufærdig.

## Dataset

Create this directory data structure.

```bash
dataset/
  images/
    train/
    val/
  labels/
    train/
    val/
```

## Træning af model

Script: train.py

**Output:** `best.pt` i `runs/detect/trainX/weights`

Replace X with actual number.

## Test af model

Script: webcam.py eller image.py
